External references used across this site or useful for self-hosted operations.

## Standards and frameworks

- [NIST SP 800-34](https://csrc.nist.gov/publications/detail/sp/800-34/rev-1/final) — Contingency Planning Guide (covers RPO/RTO concepts formally)
- [3-2-1 backup strategy](https://www.backblaze.com/blog/the-3-2-1-backup-strategy/) — Backblaze's explanation of the rule

## Backup tools

- [restic documentation](https://restic.readthedocs.io/) — encrypted, deduplicated backup tool
- [BorgBackup documentation](https://borgbackup.readthedocs.io/) — similar to restic, strong deduplication
- [Proxmox Backup Server](https://pbs.proxmox.com/docs/) — backup solution for Proxmox environments

## Monitoring and alerting

- [Prometheus](https://prometheus.io/docs/) — metrics collection and alerting
- [Grafana](https://grafana.com/docs/) — dashboards for Prometheus and other sources
- [Uptime Kuma](https://github.com/louislam/uptime-kuma) — lightweight self-hosted monitoring

## Communities

- [r/selfhosted](https://www.reddit.com/r/selfhosted/) — general self-hosting community
- [r/homelab](https://www.reddit.com/r/homelab/) — hardware and infrastructure focused
- [Homelab Wiki](https://reddit.com/r/homelab/wiki) — community-maintained starting points
